PyMunin Memcached Plugin

Multigraph Munin Plugin for monitoring Memcached Cache Server

Memcached Plugin - memcachedstats

The multigraph plugin implements the following graphs:

  • memcached_connections
  • memcached_items
  • memcached_storage
  • memcached_traffic
  • memcached_connrate
  • memcached_reqrate
  • memcached_hitpct
