PyMunin NTP Plugins

Multigraph Plugins for NTP Time Sychronization

NTP Sync Plugin - ntpstats

The multigraph plugin implements the following graphs for monitoring time synchronization status relative to active peer:

  • ntp_peer_stratum
  • ntp_peer_stats

    NTP Hosts Plugin - ntphostoffsets

    The multigraph plugin implements the following comparison graphs for a set of remote hosts with NTP Service:

  • ntp_host_stratums
  • ntp_host_offsets
  • ntp_host_delays
