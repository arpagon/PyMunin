PyMunin by aouyar

Python Module for developing Munin Multigraph Monitoring Plugins.

Multigraph Plugins for Apache Web Server, Asterisk, Disk Usage, Memcached, Network Interfaces, NTP, PostgreSQL Database, Apache Tomcat.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/aouyar/PyMunin

get the source code on GitHub : aouyar/PyMunin
